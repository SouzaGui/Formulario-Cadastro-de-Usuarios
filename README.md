# Formulario Cadastro de Usuarios

##Formulário Simples, onde é possível realizar o cadastro do usuário, atualizar, visualizar por id, deletar e visualizar todos os usuários cadastrados.

Técnologias Utilizadas:

- Java
- HTML
- CSS
- Sweetalert
- MySQL
